
# 🏠 Home Control Web App

## 📌 Project Overview

This project is a simple web-based control system where a user can control multiple devices like lights, fan, and TV from a single interface.

It simulates how real AV (Audio-Visual) systems work, where different devices are managed from one control panel.

---

## ⚙️ Features

* Turn devices ON and OFF
* Control multiple devices (Light, Fan, TV)
* Simple and clean user interface
* Real-time status update of devices
* Easy to use control panel

---

## 🧠 How It Works

* User clicks a button on the web page
* A request is sent to the backend
* Backend changes the device state (ON/OFF)
* Updated status is shown on the screen

This shows basic communication between frontend and backend, similar to real control systems.

---

## 🛠️ Technologies Used

* Frontend: HTML, CSS, JavaScript
* Backend: (Flask / Node.js — update based on your project)
* Communication: HTTP requests
